# GR経理 使い方ガイド

このシステムは、今までExcelで手作業でやっていた「通帳整理」「請求書の消込」「月別経費一覧の作成」を自動化するものです。

---

## 👉 まず知っておいてほしいこと

### このシステムは「3つのExcel作業」を1つにまとめたものです

| これまで（Excel） | このシステム |
|----|----|
| ① 銀行から通帳CSVをダウンロードして整理 | **📒 通帳タブ**に1クリックで読み込み |
| ② 届いた請求書を消込ファイルに手入力 | **📋 請求書・消込タブ**でExcel読込＋手入力 |
| ③ 通帳を見ながら月別経費一覧を作る | **📊 月別経費タブ**で自動集計（入力不要） |

### Excelと違うところ

- **保存ボタンがない** ：入力した瞬間に自動保存されます
- **数字を消すかわりに取消線** ：消込した金額は <span style="text-decoration:line-through;">¥10,000</span> のように赤い線で表示（金額は残ります）
- **遅延が色でわかる** ：前月繰越があると行が薄いオレンジ、前々月繰越があると中間色、3ヶ月以上遅れで濃い赤に

## 1. 起動と終了

### 起動のしかた

1. `GR経理/app` フォルダを開く
2. アドレスバーに `cmd` と入力してEnter（黒い画面が開きます）
3. 黒い画面で `node server.js` と入力してEnter
4. `GR経理システム起動: http://localhost:3000` と出たら成功
5. ブラウザで http://localhost:3000 を開く

### 終了のしかた

- 黒い画面で `Ctrl + C` を押す、または黒い画面を閉じる

### データはどこに保存されているか

`GR経理/app/data.db` というファイルに全部入っています。
**このファイルをUSBなどにコピーしておけばバックアップになります。**

## 2. 画面の見方

画面の一番上に5つのタブがあります。

| タブ | 何をするところ？ |
|------|------|
| 📒 通帳（銀行取引） | 銀行の通帳データを取り込んで、1件ずつ「どの経費か」を整理 |
| 📋 請求書・消込 | 支払うべき請求書を登録し、支払い終わったものに印をつける |
| 📊 月別経費（自動集計） | 通帳データから自動で作られる月別の経費一覧（見るだけ） |
| ⚙ 設定 | 会社名の登録ルール、銀行口座、施設名の管理 |
| ❓ 使い方 | このページ |

## 3. 📒 通帳（銀行取引）の使い方

### 3-1. 通帳データを読み込む

1. 「**📂 通帳データを読み込む（CSV/Excel）**」ボタンをクリック
2. 銀行からダウンロードしたCSVファイル、または親通帳Excelを選ぶ
3. プレビュー画面で中身を確認（色の意味は下記）
4. 「確定して保存」をクリック

**色の意味：**

| 色 | 意味 |
|----|----|
| 🟨 黄色 | 勘定科目が入っていない行 → あなたが入力する必要あり |
| 🟦 青色 | システムが自動で科目を入れた行 → 間違っていたら修正してください |
| ⬜ 白色 | 既にあなたが手で科目を入れた行 |

### 3-2. 1行ずつ整理する（セルをクリック）

表のセルをクリックすると、その場で編集できます。

| 編集する列 | どう入力する？ |
|----|----|
| 支払先（会社名） | 会社名を文字で入力（例：「中国電力」） |
| 勘定科目 | プルダウンから選ぶ |
| 施設名 | プルダウンから選ぶ |
| 済 | 支払い済みのチェック |
| 備考 | 何でも自由に |

### 3-3. 自動で覚えます（学習機能）

**同じ摘要の取引が出てきたら、次回からは自動で入ります。**

例：「ガス料 1234567」→ あなたが「広島ガス / 水道光熱費 / 弥山」と入力
→ 次に「ガス料 9876543」が来たら、自動で同じ情報が入ります

**間違って覚えたら？**
→ ⚙設定タブの「支払先の自動認識ルール」で修正できます

### 3-4. 支払先をまとめて設定する

「**🏢 支払先を一括で設定する**」ボタンで、似たような通帳摘要をまとめて会社名登録できます。
半角カナの読みにくい摘要をまとめて整理したい時に便利です。

### 3-5. 経費の内訳を表で見る

「**📊 経費の内訳を見る**」ボタンで、横軸：施設 × 縦軸：勘定科目 のマトリクス表示に切り替わります。

## 4. 📋 請求書・消込の使い方

これが「請求書消込ファイル」の代わりです。

### 4-1. 請求書Excelをまとめて読み込む

1. 「**📂 請求書Excelをまとめて読み込む**」ボタンをクリック
2. いつも使っている請求書一覧Excel（例：`R8.2月請求額_確認用.xlsx`）を選ぶ
3. プレビュー画面で確認
4. 「確定して保存」をクリック

**自動で読み取られる情報：**
- エンティティ（シート名から：リゾート、ほうらい、フォレスト など）
- 月（ファイル名から：「2月」など）
- 年（ファイル名の「R8」から2026年と判定）
- 業者名、勘定科目、支払方法、期日、請求先、金額、繰越分

**スキップされるシート：** 「検索」「総合計」

### 4-2. 請求書を1件だけ手で追加する

「**＋ 請求書を1件追加**」ボタンでフォームが開きます。業者名・金額・期日などを入力して保存。

### 4-3. 絞り込みで見やすく

- **エンティティタブ：** リゾート／ほうらい／フォレスト...で絞り込む
- **月タブ：** 月ごとに絞り込む
- **ステータスフィルタ：** 未払い／支払済み／一部支払い済みで絞り込む
- **業者名検索：** 業者名の一部を入れて検索

### 4-4. セルをクリックして直接編集

Excelと同じ感覚で、業者名・金額・期日など好きなセルをクリックすれば、その場で修正できます。

### 4-5. 消込する（支払い済みにする）

**これが一番大事な機能です。**

1. 支払った請求書のチェックボックスをON
2. 「**選択した請求を消込...**」ボタン
3. ダイアログで **どの金額を支払ったか** を選ぶ：
   - 今月分だけを支払い済みにする
   - 前月繰越分だけを支払い済みにする ← **「2か月前の10万円だけ払えた」ときはこれ**
   - 前々月繰越分だけを支払い済みにする ← **「3か月前の20万円だけ払えた」ときはこれ**
   - 前々々月繰越分だけを支払い済みにする
   - 全部まとめて支払い済みにする

### 4-6. 消込すると金額はどうなる？

**金額は消えません。赤い取消線が引かれるだけです。**

例：今月分 ¥10,000 を消込すると → <span style="text-decoration:line-through;color:#d63031;">¥10,000</span> のように表示

- 全部消込 → 状態が「済」に
- 一部だけ消込 → 状態が「一部」に（残りは未払い）

**間違えて消込したら？** 「**消込を取り消す**」ボタンで元に戻せます。

### 4-7. 通帳と自動で突き合わせる

「**🔄 通帳と照合して消込する...**」ボタンで、通帳の出金と請求書を自動でマッチングします。

**マッチングの仕組み：**
- 通帳の「支払先（会社名）」と請求書の「業者名」を突き合わせる
- 金額が「今月分 / 前月繰越 / 前々月繰越 / 前々々月繰越」のどれかに一致するかチェック

**使うためのコツ：**
- 事前に通帳の「支払先」が全部入っていないと照合できません（未入力のものはスキップされます）
- ⚙設定タブの「支払先の自動認識ルール」でまとめて登録しておくとスムーズ

### 4-8. 遅延が赤色でわかる

繰越が多い＝長く支払えていない請求書は、行の色が濃くなります。

| 色 | 状態 |
|----|------|
| 白 | 今月分のみ（遅延なし） |
| 🟡 薄いオレンジ | 前月繰越あり（1ヶ月遅れ） |
| 🟠 オレンジ赤 | 前々月繰越まであり（2ヶ月遅れ） |
| 🔴 濃い赤 | 前々々月繰越まである（3ヶ月以上遅れ＝深刻） |

### 4-9. 請求書を削除する

- **選択したものだけ削除：** チェック → 「選択した請求を削除」
- **表示中のものを全部削除：** 「今表示中の請求をまとめて削除...」
  - 月やエンティティで絞り込んでから実行すれば、その月・その会社の分だけまとめて削除可能
  - **再取込する前の掃除に使います**（同じ月のExcelを2回読み込むと二重になるため）

## 5. 📊 月別経費（自動集計）の見方

**ここは見るだけのページです。何も入力不要。**

通帳データから自動で集計されます：
- **横軸：** 施設（リゾート、ビュー、デルーネ...）
- **縦軸：** 勘定科目 → 支払先（会社名）
- **値：** 出金合計

### フィルタ

- 年、銀行口座、月で絞り込めます

### 表示される支払先名

- 通帳に「支払先」を入れていれば、その会社名がそのまま縦軸に並びます
- 入れていない場合は、通帳の摘要パターンがそのまま表示されます（半角カナで読みにくい）
- → **通帳タブできちんと支払先を入れておくことが大事**

## 6. ⚙ 設定の使い方

### 6-1. 支払先の自動認識ルール

通帳の「摘要」（半角カナ）から「会社名」を自動で認識するルールの一覧です。

**使い方：**
- 通帳タブで1回でも手入力すると、自動でここに登録されます
- **セルをクリックで修正・削除できます**
- 間違った学習をしたときはここで直します

**最初の1回だけやると便利：**
「**🏢 よく使う会社名をまとめて登録**」ボタンで、140件ほどの会社名（広島ガス、中国電力、サニクリーン中国など）を一括登録できます。

### 6-2. 銀行口座の登録

通帳を読み込んだ銀行口座の一覧です。
新しい銀行口座の通帳を取り込むと自動で追加されるので、基本は触らなくて大丈夫。

### 6-3. 施設・物件の登録

経費を集計するときの「施設・物件」の一覧です。
ここに登録した名前が、通帳や請求書の「施設名」プルダウンに並びます。

## 7. 月次作業の流れ（おすすめ手順）

Excelで毎月やっていた作業を、このシステムでやる場合の順番です。

### 月初めにやること

**1️⃣ 前月の通帳を読み込む**
→ 📒 通帳タブ → 「通帳データを読み込む」

**2️⃣ 通帳の「黄色い行」（未分類）を整理する**
→ 各行の「支払先」「勘定科目」「施設名」を埋める
→ 一度入れた会社は次から自動で埋まります

**3️⃣ 初回だけ：よく使う会社名をまとめて登録**
→ ⚙ 設定タブ → 「🏢 よく使う会社名をまとめて登録」

**4️⃣ 今月の請求書Excelを読み込む**
→ 📋 請求書・消込タブ → 「📂 請求書Excelをまとめて読み込む」
→ 例：`R8.〇月請求額_確認用.xlsx` を選ぶ

**5️⃣ 通帳と自動で突き合わせる**
→ 📋 請求書・消込タブ → 「🔄 通帳と照合して消込する」
→ マッチしたものを確認してまとめて消込

**6️⃣ 自動照合で拾えなかった分は手動で消込**
→ 通帳と請求書を見比べて、支払い済みのものをチェック → 「選択した請求を消込」

**7️⃣ 月別経費を確認**
→ 📊 月別経費タブ → 見るだけ、自動で集計済み

### 月内にやること

- **請求書が届いたら：** 📋 請求書・消込タブで「＋ 請求書を1件追加」
- **通帳が増えたら：** 📒 通帳タブで追加読み込み（重複は自動排除）

### 間違えて操作してしまったら

- **請求書を二重に読み込んだ** → 「今表示中の請求をまとめて削除」でその月分を消して再取込
- **間違った会社名で学習された** → ⚙ 設定 → 支払先の自動認識ルール → セルをクリックして修正
- **誤って消込してしまった** → 「消込を取り消す」ボタン

## 8. 困ったときは

### 画面が真っ白になった

1. ブラウザの「更新」ボタン（またはF5キー）
2. それでもダメなら、黒い画面で `GR経理システム起動` と出ているか確認

### ファイルのアップロードで失敗する

- Excelを**閉じた状態**でアップロードしてください（開いたままだとファイルがロックされて失敗することがあります）
- ファイルサイズは50MB以内
- 拡張子は .xlsx / .xls / .csv のみ

### 月別経費に「半角カナ」がそのまま出てくる

→ 通帳の「支払先」を入力していない取引が多いです。
⚙ 設定 → 「🏢 よく使う会社名をまとめて登録」を実行するか、通帳タブで個別に入力してください。

### 照合ボタンを押しても「0件」と出る

- 通帳の「支払先（会社名）」が空の取引はスキップされます
- 金額が完全一致しないとマッチしません（手数料・振込手数料分がずれていると不一致）
- 一括支払で複数請求が1つの振込にまとまっている場合は自動マッチできません
- これらは手動で消込してください

### データが消えた・おかしくなった

`GR経理/app/data.db` を、事前に取っておいたバックアップと入れ替えてください。
（`data.db-wal` `data.db-shm` があれば一緒に戻します）

## 付録：用語集

| 言葉 | 意味 |
|------|------|
| 摘要 | 通帳に記録されている取引の内容（半角カナが多い） |
| 支払先 | 摘要を会社名に直したもの（あなたが登録する） |
| 勘定科目 | 経費の種類（通信費・水道光熱費 など） |
| 施設 | 集計する単位（リゾート・ビュー・デルーネ など） |
| 消込 | 請求に対して支払いが済んだことを記録する作業 |
| 繰越 | 支払えずに次の月に持ち越した金額 |
| エンティティ | 会社・事業区分（リゾート・ほうらい・フォレスト など） |
| 親通帳 | 複数の銀行口座をまとめたExcel |
