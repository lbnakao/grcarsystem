1. このシステムでできること

紙でつけていた運行記録簿を、スマホやパソコンから入力できるようにしたものです。

  • 社用車の予約をとる
  • 使い終わったら「完了」ボタンを押して、距離と行先を入力する
  • 月ごとの運行記録を画面で見たり印刷したりできる

予約と「完了」の入力さえできれば、あとは自動で運行記録簿ができあがります。

2. ログインのしかた

  1. ブラウザ（Chrome・Safari など）でシステムのページを開きます
  2. 社員番号 を入力します
    （例：髙宮さんなら「101」）
  3. パスワード を入力して「ログイン」を押します

3. 予約をとる

ログインするとすぐにカレンダー画面になります。

  1. カレンダーで 使いたい日 をクリックします
  2. 予約フォームが出るので、次を入れます：
    • 車両（キャラバン／ローザ／セルボ／アトレー）
    • 開始時間と終了時間
    • 出発場所と返却場所（例：温井、いこい、FHG など）
    • 備考（任意）
  3. 「保存」を押せば予約完了です

4. 使い終わったら「完了」を押す

安芸太田町組の車両は、使い終わったら必ず「完了」ボタンを押してください。これを押さないと運行記録簿に残りません。

手順

  1. カレンダーで自分の予約をクリックします
  2. 画面の下にある ✓ 完了 ボタンを押します
  3. 入力画面が出るので、次を入れます：
    • 出庫時 積算距離 — 前回の数字がすでに入っているので、そのままでOK
    • 帰着時 積算距離 — 車のメーターの数字を見て入力
    • 使用距離 — 自動で計算されます（入力しません）
    • 行先・使用目的 — 例：「温井→広島市 資材運搬」
  4. 「完了を確定」を押します

どちらの場合も選択肢は 2つ

  • 完了 を押す → 完了画面が開くので、距離と行先を入れて確定してください
  • 完了していない を押す → お知らせを閉じます（次回ログイン時にまた出ます）
運転が終わっているのに「完了していない」を押し続けると、運行記録簿に記録されません。
運転が終わったら必ず「完了」を押してください。

6. 運行記録簿を見る・印刷する

これまで紙でつけていた運行記録簿と同じものが画面で見られます。

  1. 左メニューの 「運行記録簿」 をクリックします
  2. 車両と年月を選びます
  3. 「表示」ボタンを押します
  4. 一覧が出てきます。そのまま画面で確認できます
  5. 紙に印刷したい時は、右下の 「🖨 印刷」ボタン を押します

7. こまった時は

パスワードを忘れた

管理者に連絡してリセットしてもらってください。

間違った予約を作ってしまった

予約をクリックして、× キャンセル を押せば予約を取り消せます。

間違った距離を入れて「完了」してしまった

管理者に連絡してください。管理者が修正できます。

完了ボタンを押し忘れたまま何日か経った

次にログインした時に赤いお知らせが出ます。そこで「完了」を押して、実際の距離と行先を入れてください。
帰着時刻はその時押した時刻になりますが、運行記録簿には残ります。

動かないなど、システム自体のトラブルは管理者に連絡してください。
